# Course-Recommender
Java application recommends a student in course selection using machine learning principle.
Tools like java swing, sql and mysql are used.

Student.java takes information from student with its unique id in a java form.
In Test1.java, marks of each student in various subjects are stored in mysql database.
In Test2.java, based on Training dataset, Testing is performed and a course is recommended to a student.
File.txt contains a training dataset.
